# Project Blueprint

## Overview

This document outlines the structure, design, and features of the Flutter application. It serves as a single source of truth for the project, from its initial version to the current state.

## Style and Design

- **Theme:** The application uses a custom theme with a dark background and electric blue accents.
- **Typography:** The `google_fonts` package is used to apply the Lato font throughout the app.
- **Layout:** The layout is designed to be clean and intuitive, with a focus on readability and ease of use.

## Features

- **Home Page:** The initial screen of the app, which displays a welcome message and a button to navigate to the AI Page.
- **AI Page:** A page with a text field for user input and a button to interact with the AI.
- **Theme Management:** The `provider` package is used to manage the app's theme, allowing for easy updates and customization.
